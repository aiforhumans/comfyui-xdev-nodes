### Lazy Loading Pattern
```python
def get_torch():
    """Lazy torch import to improve startup time"""
    global _torch_cache
    if _torch_cache is None:
        try:
            import torch
            _torch_cache = torch
        except ImportError:
            _torch_cache = False
    return _torch_cache if _torch_cache is not False else None
```
