## Usage Tips

### Combining Elements
- Mix 3-5 lighting types for complex scenes
- Combine 2-3 pose elements for natural positioning
- Blend multiple quality enhancers (avoid over-saturation)
- Use negative prompts sparingly with SDXL

### Weighting Techniques
```sd
(cinematic lighting:1.3), (professional portrait:1.2), soft shadows, natural pose
```
```sd
dramatic lighting:0.8, studio quality, clean background, (perfect skin:1.1)
```

### Resolution Guidelines
- 512x512: Focus on subject and lighting
- 768x768: Add environment details
- 1024x1024: Include texture and atmosphere
- 1024x1536: Full body and environment

### Style Mixing
Start with a base style (photographic, artistic, etc.) then add specific elements:
```sd
photographic portrait, (cinematic lighting:1.2), professional quality, dramatic shadows
```
